## 리액티브 프로그래밍이란



**리액티브 프로그래밍**은 데이터 스트림과 변경 사항 전파를 중심으로 하는 비동기 프로그래밍 패러다임

**리액티브 프로그램**은 데이터와 데이터 스트림에 영향을 미치는 모든 변경 사항을 관련된 모든 당사자들에게 전파하는 프로그램



## 함수형 리액티브 프로그래밍을 적용해야하는 이유

- 콜백 지옥의 제거
- 오류 처리를 위한 표준 메커니즘
- 간결해진 스레드 사용
- 간단한 비동기 연산
- 전체를 위한 하나, 모든 작업에 대해 동일한 API
- 함수형 접근
- 유지 보수 가능하고 테스트 가능한 코드
